# NeuroMuscleAI-MVP — Demo

## 中文（简要）

欢迎使用 NeuroMuscleAI-MVP。本页面展示了如何快速运行 demo：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

示例图片与说明位于仓库 `docs/` 目录。
